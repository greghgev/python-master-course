# Activity 01 – Grade Average Calculator

## Objective
Develop a structured Python program that:

- Allows users to input subjects and grades (0–10)
- Calculates overall average
- Determines passed/failed subjects (threshold 5.0)
- Identifies highest and lowest grades
- Includes input validation
- Uses at least 3 functions (no OOP)

## Execution
python calculadora_promedios.py
